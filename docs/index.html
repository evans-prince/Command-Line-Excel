Commands (raw)

Click a cell to select it, then use the top field to set its value or formula (arithmetic like 3+4 or a reference like B1 also works). The second field sends a raw command straight to the C engine, exactly like the native CLI's stdin — try function calls, ranges, or scroll_to<cell>. Only ten rows/columns are visible at a time, matching the original terminal UI; scroll with the ▲▼◀▶ buttons or w/a/s/d in the command field. A div-by-zero shows as ERR, same as the terminal version. Quitting is disabled here so the sheet stays alive in your tab.
